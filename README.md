# VB 2026 Playoff PWA

Frissített, iPhone-ra optimalizált statikus PWA a 10 meccses JSON alapján.

## Használat

1. Csomagold ki a ZIP-et.
2. Töltsd fel a tartalmát egy GitHub repository gyökerébe.
3. Kapcsold be a GitHub Pages publikálást.

Az app build step nélkül fut, az adatforrás az `assets/app-data.json`.
